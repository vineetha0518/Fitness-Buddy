AGENT_INSTRUCTIONS — Customise Without Touching Core Code

Edit only AGENT_INSTRUCTIONS in services/fitness_agent.py to change the AI's behaviour:

KeyWhat it controls
"name"AI assistant's name (default: FitBot)
"personality"Core character and communication tone
"motivation_style"How it motivates users, cultural phrases
"specialization"Fitness expertise area and training style
"nutrition_focus"Indian food preferences, regional meals
"intensity_guidance"Workout intensity rules and progression
"language_tone"Formality, jargon level, Hindi phrases
"response_format"Emoji usage, bullet points, Pro Tips
"safety_guidelines"Medical disclaimers (non-negotiable)
"scope"Topic boundaries for the assistant

Quick Start

1
Install dependencies
cd FitnessBuddy
pip install -r requirements.txt
2
Configure IBM credentials
IBM_API_KEY=your_ibm_cloud_api_key
IBM_PROJECT_ID=your_watsonx_project_id
IBM_WATSONX_URL=https://us-south.ml.cloud.ibm.com
IBM_MODEL_ID=ibm/granite-3-8b-instruct
3
Run the app
python app.py
# Open http://localhost:5000
4
Deploy to Render (free)
Build: pip install -r requirements.txt
Start: gunicorn app:app
